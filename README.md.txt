# Internship Tracker DBMS Project

A simple database system to manage and track internship applications.

## Features
- Store internship company details
- Track application status
- Record application dates
- Maintain student information

## Technologies Used
- SQL (MySQL)
- Git & GitHub for version control

## Database Tables
1. Students
2. Companies
3. Applications

## Example Use Case
A student can track where they applied for internships, the company name, role, and application status.
